# InvoiceKit — Invoice Generator for Freelancers

Generate professional invoices with PDF export.

**Live:** https://invoicekit-olive.vercel.app

## Features
- Create and customize invoices
- PDF export
- Client management
- Line items with calculations

## Tech Stack
React 19 · TypeScript · Tailwind CSS v4 · Vite

## Getting Started
```bash
npm install
npm run dev
```
